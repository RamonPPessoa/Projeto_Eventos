
<p align="justify"> Desenvolvimento de uma aplicação  de eventos utilizando o framework php Laravel, com o obejtivo de aprendizado e dominio da linguagem php e do framework Laravel. A finalidade da aplicação é cadastrar, criação, cancelar um evento, editar e buscar os eventos existentes e ou cadastrados.</p>
